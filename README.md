# 用户认证识别服务

这是一个基于 Spring Boot 的用户认证与识别服务项目，实现了登录、注册等核心功能。

## 技术栈

- **后端**: Spring Boot, MyBatis
- **前端**: HTML, CSS, JavaScript
- **数据库**: MySQL

## 功能特性

- ✅ 用户登录
- ✅ 用户注册
- ✅ 跨域请求处理 (CORS)
